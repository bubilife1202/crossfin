# CrossFin As-Is / To-Be 분석

---

## AS-IS: 현재 상태

### 한 줄 요약

**무료 공개 API를 x402 결제로 감싼 프록시 서버. 사용자 0명.**

### 30개 엔드포인트 실태

| 분류 | 개수 | 설명 |
|------|------|------|
| 무료 API 프록시 (가치 없음) | 22개 | Bithumb/Upbit/Coinone/Naver를 그대로 전달. 에이전트가 직접 호출하면 무료 |
| 편의성 있음 (약한 가치) | 5개 | 5거래소 비교, 아비트리지 판단 등. 여러 API 묶음 + 간단한 수학 |
| 고유 가치 있음 | 2개 | 김치프리미엄 히스토리(자체 DB), 시스템 리포트(내부 데이터) |
| 의미 없음 | 1개 | Enterprise($20) — UUID만 생성하고 끝 |

### 구체적 예시

**$0.01 내고 환율 조회** → `open.er-api.com` 무료 호출하면 같은 데이터
**$0.02 내고 Upbit 시세** → `api.upbit.com` 무료 호출하면 같은 데이터
**$0.03 내고 한국 뉴스** → Google News RSS 무료 파싱하면 같은 데이터
**$0.05 내고 KOSPI 지수** → Naver Finance API 무료 호출하면 같은 데이터

### 가치가 있는 2-3개

1. **아비트리지 판단 서비스** ($0.10) — 5거래소 호가창 + 슬리피지 + 변동성 + 전송시간을 종합해서 EXECUTE/WAIT/SKIP 판단. 이건 단순 프록시가 아님.
2. **크로스 거래소 비교** ($0.08) — 4개 국내거래소 + 글로벌 가격 비교 후 ARBITRAGE/MONITOR/HOLD 판단.
3. **김치프리미엄 히스토리** ($0.05) — 매시간 자동 스냅샷. 축적된 데이터는 CrossFin만 가지고 있음.

### ✅ 구현 완료: TIER 1/2 번들 엔드포인트 (v1.8.0, 2026-02-18)

AS-IS에서 지적한 "직접 호출하면 무료인데 왜 돈을 내나" 문제의 해결:

| 엔드포인트 | 가격 | 합성하는 개별 API | 개별 합산 비용 | 절감 |
|-----------|------|------------------|--------------|------|
| **Morning Brief** `/api/premium/morning/brief` | $0.20 | 김프+FX+지수+모멘텀+뉴스 (5개) | $0.17 | 원콜 편의성 + 속도 |
| **Crypto Snapshot** `/api/premium/crypto/snapshot` | $0.15 | 5거래소+김프+볼륨+FX (4개) | $0.17 | 원콜 편의성 + 절감 |
| **Kimchi Stats** `/api/premium/kimchi/stats` | $0.15 | 김프+아비트리지+크로스거래소 (3개) | $0.23 | 35% 절감 + 트렌드 분석 |
| **Stock Brief** `/api/premium/market/korea/stock-brief` | $0.10 | 종목상세+뉴스+수급+공시 (4개) | $0.16 | 37% 절감 + 원콜 |

핵심: 에이전트가 개별 API 4-8번 호출 대신 번들 1번 호출 → 비용↓ 속도↑ 코드↓

### 핵심 문제

에이전트가 CrossFin을 쓸 이유가 없다. 직접 호출하면 무료인데 왜 돈을 내나.

## TO-BE: 되어야 할 상태

### 한 줄 목표

**에이전트가 직접 할 수 없는 분석/판단/실행을 해주는 서비스. 쓰는 에이전트가 있는 서비스.**

### 방향 1: 모든 엔드포인트에 "판단 레이어" 추가

지금 가치 있는 엔드포인트는 EXECUTE/WAIT/SKIP 같은 판단을 해주는 것들. 이걸 모든 엔드포인트에 적용.

| As-Is | To-Be |
|-------|-------|
| 김치프리미엄 3.2% (숫자) | "BTC 김치프리미엄 3.2%. Bithumb 매도 → Binance 매수 시 수수료/전송비 차감 후 순이익 $47. 슬리피지 위험 낮음. **EXECUTE** (confidence 87%)" |
| KOSPI 2,847 (숫자) | "KOSPI 2,847 (+1.2%). 외인 3일 연속 순매수 +2,340억. 기관 순매도. 반도체 테마 +3.8% 주도. **단기 상승 모멘텀** (confidence 72%)" |
| Upbit BTC 시세 (숫자) | "BTC 92,340,000원. 24h 거래량 상위 15%. 5거래소 중 Upbit이 최저가. Bithumb 대비 0.12% 저렴. **Upbit 매수 권장**" |
| 한국 뉴스 헤드라인 (목록) | "삼성전자 반도체 수출 +23% — 관련 종목 SK하이닉스/한미반도체 수급 분석: 외인 순매수 전환. **투자 시그널: 긍정적**" |

**핵심**: raw 데이터는 무료. **판단은 유료**. 에이전트가 5개 API를 호출하고 분석하는 걸 CrossFin이 한 번에 해주면 가치가 생김.

### 방향 2: 한국 시장 특화 "에이전트 전용 인텔리전스"

해외 에이전트가 한국 시장 데이터를 영어로 받을 수 있는 유일한 API가 되면 독점적 가치.

| 새 엔드포인트 (예시) | 가치 |
|-------------------|------|
| `/api/premium/korea/market-brief` | 한국 시장 전체 요약 (주식+크립토+FX+뉴스) 한 번에 |
| `/api/premium/korea/arbitrage-alert` | 김치프리미엄 + 5거래소 + 환율을 종합한 실시간 기회 알림 |
| `/api/premium/korea/stock-radar` | 외인/기관 수급 급변 + 테마 급등 + 공시 이벤트를 종합한 종목 스캐너 |
| `/api/premium/korea/sentiment` | 네이버 뉴스 + 거래량 + 수급을 종합한 시장 심리 지수 |

이것들은 에이전트가 직접 만들려면 여러 API + 분석 로직이 필요 → CrossFin이 해주면 가치 있음.

### 방향 3: 실제 에이전트 사용 증명

VibeLabs가 보는 건 "작동하는 프로덕트". 코드만 있고 쓰는 사람이 없으면 의미 없음.

| 할 일 | 효과 |
|--------|------|
| Claude Desktop + MCP로 실제 시나리오 데모 | "에이전트가 CrossFin으로 돈 내고 데이터 사는" 장면 |
| 데모에서 x402 결제 → Basescan 확인 | 온체인 증거 |
| live.crossfin.dev에 실제 활동 로그 | 빈 대시보드가 아닌 살아있는 서비스 |
