## 1. 시장 규모 & 타겟 유저

### 한국 가상자산 투자자 현황
- **총 투자자 수**: 약 1,560만 명 (2024년 11월 기준, 한국은행 자료)
- 7월 1,474만 -> 8월 1,482만 -> 9월 1,488만 -> 10월 1,498만 -> 11월 1,559만 (월 10~60만 명 증가 추세)
- **일평균 거래대금**: 14.9조원 (코스피 9.9조 + 코스닥 6.9조와 맞먹는 규모)
- **시가총액 기준 보유액**: 102.6조원 (2024년 11월)
- 데이터 출처: 업비트, 빗썸, 코인원, 코빗, 고팍스 5개 거래소 합산

### 한국 텔레그램 사용자 현황
- **한국 MAU**: 약 347만 명 (2024년 8월, 역대 최대치)
- 전월(316만) 대비 9.8% 증가, 2021년 3월 이후 최대 증가폭
- **글로벌 MAU**: 10억 명 돌파 (2025년 3월, 파벨 두로프 발표)

### 텔레그램 사용 비율 추정 (크립토 투자자 대상)
| 구분 | 수치 | 비고 |
|------|------|------|
| 한국 크립토 투자자 총 수 | 1,560만 명 | 2024년 11월 기준 |
| 한국 텔레그램 전체 MAU | 347만 명 | 2024년 8월 기준 |
| 크립토 투자자 중 텔레그램 사용 추정 | 150~200만 명 | 텔레그램 사용자의 약 45~58%가 크립토 관련 |
| 크립토 투자자 대비 텔레그램 사용률 | **약 10~13%** | 1,560만 명 중 150~200만 명 |

**근거**: 한국 텔레그램은 크립토 커뮤니티에서 핵심 소통 채널. 디스프레드 조사(2023)에서 110개 주요 크립토 텔레그램 채널의 메시지를 분석한 결과, 코인니스가 X(트위터)에 이어 2위 정보 출처로 확인됨. 한국에서 텔레그램은 크립토 외 일반 메신저 용도로는 거의 사용되지 않으므로, MAU의 상당 부분이 크립토 관련 사용자로 추정.

## 4. CrossFin 차별화 포인트

### 4-1. CrossFin의 기존 강점
- **5개 국내 거래소** 실시간 데이터 (빗썸, 업비트, 코인원, 고팍스 + 코빗)
- **3개 글로벌 거래소** 실시간 데이터 (바이낸스, OKX, Bybit)
- 김치프리미엄 계산 API **이미 동작 중**
- A2A/MCP 표준 지원으로 AI 에이전트 연동 가능

### 4-2. 경쟁 우위 차별화 전략

| 차별화 포인트 | 설명 | 경쟁사 대비 |
|---------------|------|------------|
| **1. 멀티 거래소 김프** | 5개 국내 + 3개 글로벌 = 최대 15개 거래소 쌍 김프 비교 | 더따리: 3개 국내, 김프봇: 업비트만 |
| **2. 거래소별 최적 아비트리지 경로** | "지금 어디서 사고 어디서 팔면 최대 수익?" 자동 추천 | 경쟁사 없음 (수동 비교만 제공) |
| **3. 실시간 알림 커스터마이징** | 코인별/거래소 쌍별/임계값별 세밀한 알림 설정 | 더따리: 기본 김프 알림만, 개인봇: 제한적 |
| **4. 역프 전략 알림** | 역프(마이너스 김프) 발생 시 해외 매도 -> 국내 매수 기회 알림 | 대부분 김프만 집중, 역프 간과 |
| **5. AI 기반 김프 예측** | 과거 데이터 기반 김프 추세 예측 (상승/하락 확률) | 경쟁사 전무 |
| **6. 포트폴리오 김프 추적** | "내가 보유한 코인들의 김프 합산 추적" | 경쟁사 전무 |
| **7. API + 봇 번들** | 개인은 텔레그램 봇, 기관은 API 직접 연동 | 더따리: API 없음, 김프가: API 없음 |
| **8. 고팍스 포함** | 고팍스 데이터를 포함하는 유일한 서비스 (틈새 차익 기회) | 대부분 업비트/빗썸만 |

### 4-3. 킬러 피처 제안

1. **"김프 스나이퍼" 알림**: 특정 코인의 김프가 사용자 설정 임계값 돌파 시 즉시 알림 + 최적 거래 경로 안내
2. **"역프 헌터" 모드**: 역프 발생 코인 실시간 스캔 + 해외->국내 차익거래 기회 알림
3. **"김프 히트맵"**: 전체 코인의 김프 현황을 한눈에 볼 수 있는 비주얼 대시보드 (텔레그램 미니앱)
4. **"거래소 스프레드"**: 국내 거래소 간 가격차도 추적 (업비트 vs 빗썸 vs 코인원)

## 6. 요약 & 권장 전략

### 시장 기회
- **TAM**: 한국 크립토 투자자 1,560만 명
- **SAM**: 텔레그램 사용 크립토 투자자 약 150~200만 명
- **SOM**: 김프 관심 + 유료 의향 사용자 약 3~5만 명 (SAM의 2~3%)

### 경쟁 환경 요약
- **직접 경쟁자 부재**: 텔레그램 전문 김프 알림봇으로 유료 서비스를 운영하는 곳이 거의 없음
- **더따리**가 가장 가까운 경쟁자이나, 웹/앱 중심이고 텔레그램은 보조 채널
- 대부분의 김프 서비스가 **무료 + 광고/레퍼럴** 모델 -> 유료 전환 기회
- 개인 개발 봇은 안정성/기능이 부족 -> **전문 서비스 수요 존재**

### 권장 실행 전략
1. **MVP**: BTC/ETH/XRP 등 주요 10개 코인 김프 알림봇 (Free + Basic 플랜)
2. **차별화**: 멀티 거래소 비교 + 최적 아비트리지 경로 (Pro 플랜)
3. **확장**: AI 예측 + 포트폴리오 추적 (Premium 플랜)
4. **마케팅**: 크립토 텔레그램 채널 광고 + 커뮤니티 바이럴 + 거래소 콜라보
5. **부수 수익**: 거래소 레퍼럴 + B2B API 판매 동시 추진

### 예상 1년차 시나리오 (보수적)
- **무료 사용자**: 5,000~10,000명
- **유료 전환**: 500~1,000명 (10% 전환율)
- **월 매출**: 445~890만원 (구독) + 150~500만원 (레퍼럴/광고)
