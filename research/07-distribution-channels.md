## 시장 개요

- 한국 가상자산 투자자: **1,600만+ 계좌** (2025년 기준, 주식 투자자 수 초과)
- 5대 원화 거래소(업비트, 빗썸, 코인원, 코빗, 고팍스) KYC 인증 약 627만 명
- 한국은 전 세계에서 가장 활발한 리테일 크립토 시장 중 하나
- 정보 소비 주요 플랫폼: 텔레그램(1위) > X(트위터) > 유튜브 > 네이버 카페 > 디시인사이드

## 9. 초기 무비용 마케팅 전략 (커뮤니티 씨딩)

### Phase 1: 기반 구축 (1~2주)

1. **CrossFin 텔레그램 채널 개설**
   - 김치프리미엄 실시간 알림 무료 제공
   - 급등/급락 알림 (업비트 vs 바이낸스 가격차 5% 이상 시 자동 알림)
   - 목표: 첫 2주 500명 확보

2. **X(트위터) 계정 활성화**
   - 매일 김치프리미엄 차트 자동 게시 (봇 연동)
   - #김치프리미엄 #KimchiPremium 해시태그 활용
   - 한국 크립토 트레이더 팔로우 & 인용 RT

3. **GitHub 정비**
   - crossfin SDK README 한국어/영어 완비
   - 사용 예제 코드 충실화
   - "Good First Issue" 라벨 추가

### Phase 2: 씨딩 확산 (2~4주)

4. **네이버 카페 씨딩**
   - 비트맨 카페: "김프 실시간 확인하는 텔레그램 봇 만들어봤습니다" 형태 게시
   - 코인판: "개발자가 만든 김프 알림 봇 무료 배포합니다" 게시
   - 자연스러운 사용 후기 형태 필수 (광고 티 최소화)

5. **개발자 커뮤니티 진출**
   - GeekNews: "한국 크립토 데이터 API를 만들었습니다" Show HN 스타일
   - Disquiet: 프로덕트 론칭 게시
   - Velog: "김치프리미엄 알림 봇 만들기 - CrossFin API 활용" 튜토리얼

6. **텔레그램 채널 씨딩**
   - 코인같이투자, 마샬 공유방 등 주요 채널에 자연스러운 소개
   - "이 봇 써봤는데 김프 알림 꽤 정확합니다" 형태

### Phase 3: 바이럴 루프 (4~8주)

7. **무료 체험 → 프리미엄 전환**
   - 기본 알림 무료, 고급 기능(다중 코인, 커스텀 임계값) 유료
   - "친구 초대하면 프리미엄 1주 무료" 레퍼럴 프로그램

8. **콘텐츠 마케팅**
   - 주간 김치프리미엄 리포트 발행 (데이터 기반)
   - X에서 바이럴 데이터 카드 (인포그래픽)
   - "김프가 10% 넘었던 날들" 같은 역사적 데이터 콘텐츠

9. **유튜버 리뷰 유도**
   - 소형 유튜버(1~5만 구독자)에게 무료 프리미엄 제공 → 리뷰 요청
   - 라이브 방송 중 김프 알림 시연 요청

## 핵심 인사이트 & 권장사항

### CrossFin에 가장 적합한 채널 (우선순위)

1. **텔레그램** — 핵심 플랫폼. 한국 크립토 투자자의 1차 정보원. 자체 채널 필수.
2. **X(트위터)** — 데이터 시각화 콘텐츠로 자연 바이럴 유도. 무비용 최대 효과.
3. **개발자 커뮤니티** — API/SDK 유료 고객 직접 접근. 전환율 최고.
4. **네이버 카페** — 리테일 투자자 100만+ 접근. 씨딩 주의 필요.
5. **유튜브** — 비용 발생하지만 인지도 확대에 효과적. 2단계 이후 투자.

### 한국 시장 특수성
- 카카오톡(KakaoTalk)도 크립토 커뮤니티 형성에 사용되나, 폐쇄형이라 씨딩 어려움
- 한국 투자자는 단기 가격 변동에 민감 → "김프 알림"은 정확히 이 니즈 타겟
- 투기적 성향이 강해 "돈이 되는 정보"로 어필해야 효과적
- TokenPost, Block Media 등 한국 크립토 미디어 PR도 신뢰도 구축에 효과적
