## 2. Grab Financial 분석

### 회사 개요
- 동남아 최대 슈퍼앱 (라이드헤일링, 음식배달, 결제)
- Grab Financial은 2018년 출범한 금융 서비스 부문

### GrabPay 현황 (중요 변화)
> **GrabPay는 2026년 1월 15일부로 서비스 종료됨. 이후 새로운 결제 건 생성 불가.**

이는 CrossFin의 GrabPay 직접 연동 전략에 큰 영향을 미침.

### 개발자 생태계
- **개발자 포털**: developer.grab.com
- **SDK 지원 언어**: Go, PHP, Node.js, Python, Java, .NET
- GrabPay Merchant Integration SDK: 인증 헤더 생성, 결제 상태 조회, 다중 리전 지원

### API 연동 경로 (GrabPay 종료 이후)
GrabPay 직접 연동은 불가하나, 다음 PSP를 통한 간접 연동은 가능했음:
- **Adyen**: GrabPay API-only 연동
- **PayPal**: GrabPay 대체 결제 수단
- **2C2P**: SDK 기반 연동

### AI 투자 동향
- 2025년 4월: OpenAI, Anthropic 협력으로 **AI Merchant Assistant** 및 **AI Driver Companion** 출시
- 에이전트 기반 상거래 가능성 존재

### CrossFin 시사점
- GrabPay 종료로 직접 결제 연동은 **불가**
- Grab의 AI 머천트 도구와의 **에이전트-투-에이전트** 연동 가능성 탐색 필요
- Grab 생태계 내 **GrabFood/GrabMart API** 등 비결제 API 연동은 여전히 가능

## 3. GCash (필리핀)

### 시장 지위
| 지표 | 수치 |
|------|------|
| 시장 점유율 | **89%** (모바일 월렛 시장) |
| 활성 사용자 | 8,100만 (2025.01) |
| 가맹점 | 250만+ |
| 모기업 | Globe Telecom (Mynt) |

### API 연동 현황
GCash는 직접 개방형 API를 제공하지 않으며, 연동은 다음 경로로 가능:
1. **직접 파트너십**: GCash 파트너 온보딩 프로세스 통해 접근
2. **PSP/게이트웨이 경유**:
   - 2C2P PGW SDK
   - Checkout.com (API-only)
   - ECOMMPAY
   - PayRex (원API 다중결제)
   - HitPay (RESTful API)
3. **결제 플로우**: 결제 생성 -> GCash UI 리다이렉트/핸드오프 -> 웹훅/콜백 확인

### 크립토 연동 (핵심!)
- **2025년 9월**: USDC(서클) 지원 시작 - 사용자가 앱 내에서 달러 페그 스테이블코인 보유 가능
- **대상 규모**: ~1억 사용자에게 USDC 접근성 제공
- Web3, NFT, P2E(Play-to-Earn) 시장 진출 의지 표명
- 크립토 제품의 **UX 간소화** 방향 추구

### 필리핀 디지털 월렛 시장 전망
- 2025년 $4.1B -> 2035년 $11.85B (CAGR 11.2%)
- 디지털 자산의 금융 주류 편입 가속 (송금, 급여, 투자 헤징)

### CrossFin 시사점
- GCash의 USDC 지원 = **크립토-피아트 브릿지** 기회
- 필리핀 OFW(해외근로자) 송금 시장과 크립토 결합 가능성
- PSP 경유 연동이 현실적 접근법
- **89% 시장 점유율**로 필리핀 진출 시 GCash는 필수 파트너

## 4. GoPay / DANA (인도네시아)

### 인도네시아 시장 개요
| 지표 | 수치 |
|------|------|
| 모바일 결제 시장 (2026) | **$48.39B** |
| 2031년 전망 | $98.87B |
| CAGR | 15.36% |
| 상위 5개 월렛 시장 점유율 | ~70% (개별 25% 미만) |

### GoPay
- Gojek 슈퍼앱 내 통합 + **독립형 앱** 출시로 범위 확대
- 기능: 계좌이체, 은행 송금, 공과금 납부, 모바일 충전, 가맹점 결제
- **QRIS 지원**: 국가 표준 QR 결제 완전 호환

### DANA
- 2025년 초 **1.4억 사용자** 돌파
- 소액 저축, 공과금 납부 등 농촌 대상 기능 강화
- QRIS 연동으로 전국 가맹점 결제 지원
- **크로스보더 결제**: 말레이시아, 싱가포르, 태국 지원

### QRIS (Quick Response Indonesia Standard) - 핵심 인프라
| 지표 | 수치 |
|------|------|
| 사용자 수 | 5,700만+ |
| 가맹점 수 | 3,800만+ |
| 2025 Q1 거래 건수 | 26억 건 |
| YoY 성장률 | 175-226% |

**크로스보더 QRIS 현황:**
- 운영 중: 태국, 말레이시아, 싱가포르
- 2025년 8월 출시: **일본** (최초 비ASEAN 국가)
- 시범 중: **중국** (PBoC와 협력, 2025년 말 운영 목표)

### API 접근성
- QRIS 통합으로 단일 QR 코드로 모든 인도네시아 결제 수단 수용
- 주요 연동 PSP: HitPay, Nuvei, TransFi
- TransFi: 단일 API로 인도네시아 모바일 월렛 연동, 현지 통화 수취, 글로벌 정산

### CrossFin 시사점
- QRIS 표준이 **단일 연동 포인트** 역할 -> 개별 월렛 연동 불필요
- 인도네시아는 **동남아 최대 시장** ($48.4B)
- QRIS 크로스보더 확장이 일본-한국 연결 가능성 제공
- DANA의 크로스보더 기능은 동남아 내 다중 국가 결제 라우팅에 활용 가능

### LINE Pay 대만 (핵심 시장)
| 지표 | 수치 |
|------|------|
| 사용자 수 | **1,360만** (대만 인구의 ~50%) |
| 누적 거래액 (2025.11) | NTD 7,910억 ($25.2B) |
| 해외 거래액 전망 (2025) | NTD 15억 ($47.8M) |
| AI 머천트 도구 | 출시됨 |

### 크로스보더 결제 동맹
LINE Pay는 크로스보더 모바일 결제 동맹을 구축:
- **LINE Pay Japan** (종료됨)
- **Rabbit LINE Pay** (태국)
- **Naver Financial** (한국)
- **NHN PAYCO** (한국)

사용자들이 파트너 서비스 국가에서 크로스보더 결제 가능.

### Rabbit LINE Pay (태국) API
- **개발자 포털**: developers-pay.line.me
- 연동 방식: Omise.js, 모바일 SDK (iOS/Android), 직접 API
- 가맹점 등록 시 Integration Key + Test Key 제공
- PSP 경유: Airwallex, Omise, Antom, Nuvei

### 기능
- 온/오프라인 결제
- LINE 사용자 간 송금
- 공과금 납부, 고속도로 요금, 영화 티켓

### CrossFin 시사점
- 일본 LINE Pay 종료 -> 일본 시장은 다른 채널 필요
- 대만 LINE Pay의 **50% 인구 침투율**은 대만 진출 시 핵심 파트너
- **Naver Financial 동맹**: 한국-대만-태국 크로스보더 결제 경로 활용 가능
- Rabbit LINE Pay는 태국 진출 시 주요 연동 대상

- 온체인 Proof of Reserves(PoR) 투명성 제공
- 주문 유형: 지정가, 시장가, 스톱-리밋

**김치 프리미엄 (한국)**:
- 2025년 7-8월: 역사적 **역전** 발생 (BTC가 한국에서 2% 할인)

- 2026년 1월: 0.61% 프리미엄 (Upbit ₩134.03M vs Binance ₩133.22M)
- KOSPI 70%+ 급등으로 크립토에서 주식으로 자금 이동
- 2026년 상반기 FIU 제재 시행 전까지 프리미엄 0% 근처 유지 전망

**동남아 프리미엄**:
- 인도네시아, 필리핀, 태국 등에서의 로컬 거래소 프리미엄 데이터는 제한적
- 다만 자본 통제가 있는 국가(인도네시아, 필리핀)에서는 구조적 프리미엄 발생 가능성 존재
- 크로스보더 크립토 차익거래는 각국 규제 및 출금 제한으로 실행 난이도 높음

### CrossFin 시사점
- 김치 프리미엄 축소 -> 한국-글로벌 차익거래 기회 감소
- 동남아 로컬 거래소 가격 모니터링 데이터 서비스 가능성
- GCash USDC 지원 + 필리핀 크립토 시장 성장 = **필리핀 크립토-피아트 브릿지** 유망

## 8. CrossFin 아시아 확장 우선순위

### 확장 경로: 한국 -> 일본 -> ?

#### Tier 1: 싱가포르 (최우선)

**이유:**
1. **핀테크 허브**: MAS의 명확한 규제, 글로벌 기업 집중
2. **게이트웨이 역할**: 싱가포르 진출 = 동남아 전체 접근성 확보
3. **크립토 친화적**: 양도소득세 없음, DSTSP 라이선스 체계 정비
4. **Project Nexus**: MY, TH, ID, PH와의 크로스보더 결제 인프라 참여
5. **한국 연결**: KRW 스테이블코인 -> SGD 결제 경로 구축 가능

**예상 비용**: MPI 라이선스 S$250,000+ 자본금

#### Tier 2: 인도네시아 (최대 시장)

**이유:**
1. **동남아 최대 디지털 결제 시장** ($48.4B)
2. **QRIS 표준**: 단일 연동으로 전체 생태계 접근
3. QRIS 크로스보더가 일본, 한국 연결 가능성 (향후)
4. 1.4억 DANA 사용자, 9.6M Indodax 사용자 등 대규모 TAM

**리스크**: OJK 전환기 규제 불확실성, 높은 진입 장벽

#### Tier 3: 필리핀 (크립토-피아트 브릿지)

**이유:**
1. **GCash 89% 독점** = 단일 파트너로 시장 커버

## 10. 핵심 발견 & 전략적 인사이트

### GrabPay 종료의 의미
GrabPay가 2026년 1월 종료됨에 따라, Grab 생태계 직접 결제 연동은 불가. 그러나 Grab의 AI 도구(OpenAI/Anthropic 협력)는 에이전트 기반 상거래의 가능성을 시사.

### QRIS가 게임 체인저
인도네시아 QRIS는 단일 표준으로 전체 결제 생태계를 통합. 크로스보더 확장(일본, 중국)은 CrossFin의 아시아 결제 라우팅 비전과 직접 연결.

### GCash + USDC = 킬러 조합
필리핀 GCash의 USDC 지원은 크립토-피아트 브릿지를 사실상 완성. 1억 사용자 기반의 스테이블코인 접근성은 CrossFin의 크로스보더 크립토 결제 전략에 핵심적.

### 김치 프리미엄 축소 → 동남아 프리미엄으로 전환
한국 김치 프리미엄이 0-1%로 축소되면서 차익거래 기회 감소. 그러나 동남아 각국의 자본 통제로 인한 로컬 프리미엄 모니터링은 새로운 데이터 서비스 기회.

### 싱가포르는 거점, 인도네시아는 볼륨, 필리핀은 크립토 브릿지
- **싱가포르**: 법인 + 라이선스 거점, 동남아 전체 게이트웨이
- **인도네시아**: 거래량과 사용자 수 기반의 스케일
- **필리핀**: 크립토-피아트 전환의 최적지
