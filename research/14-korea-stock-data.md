### 1.3 외인/기관 수급 데이터를 영어로 제공하는 곳

**사실상 전무하다.** 유일한 경쟁자는 **TradePulse**로, Datarade를 통해 한국 시장 실시간 투자자 유형별 주문흐름 데이터를 판매 중이다:
- 가격: **월 $10,000 ~ 연 $120,000**
- 정확도: 외국인 기관 분류 89%, 기관 분류 79%
- KRX 직접 데이터 수신, AI 기반 분석
- KOSPI 시총 2,000억+ / KOSDAQ 시총 1,500억+ 약 1,000종목 커버

**핵심 인사이트**: TradePulse가 월 $10,000에 파는 데이터를 CrossFin이 훨씬 저렴하게 제공할 수 있다면, 거대한 시장 기회가 존재한다.

## 2. 수요 분석

### 2.1 한국 주식 시장의 외국인 투자 규모

- **외국인 보유액**: 2025년 12월 기준 **1,326.8조원 (약 $9,183억)** — 시가총액의 **30.8%**
- **2025년 주요 자금 흐름**:
  - 2025년 상반기: 약 $105억 순매도
  - 2025년 하반기: 약 $92억 순매수 (조사 대상 전체 시장 중 최대 유입)
  - 2025년 12월: 1.52조원 순매수
- **지역별 분포 (2025.12)**: 유럽 1.6조원 > 미주 4,000억원 > 아시아 3,000억원
- **국가별**: 프랑스 1조원, 영국 8,000억원 순매수 선두

### 2.2 KOSPI 2026년 랠리와 글로벌 관심 급증

**KOSPI는 역대급 랠리 중이다:**
- 2025년 **76% 상승**, 2026년 2월 기준 **5,808.53** (사상 최고)
- 삼성전자(+130%), SK하이닉스(+278%) — 글로벌 AI/HBM 핵심 공급사
- 삼성+SK하이닉스 시총 비중 **35.95%** (사상 최고)
- **블랙록, SK하이닉스 5% 지분 공시** — 글로벌 기관 관심의 상징
- 외국인 순매수가 KOSPI 4,400 돌파 견인 (하루 2.2조원 순매수)

### 2.3 EWY ETF 투자자 수요

- **iShares MSCI South Korea ETF (EWY)**: 한국 주식 접근 가장 인기 있는 수단
- **1년 수익률**: 148.91% (배당 포함)
- **1년 순유입**: **$47.9억** (약 6.3조원) — 폭발적 수요 증가
- 운용보수 0.59%, 배당수익률 1.51%
- **EWY 투자자가 필요로 하는 데이터**:
  - 삼성/SK하이닉스 등 핵심 보유 종목 외인/기관 수급
  - KOSPI 실시간 동향과 뉴스 (영어)
  - 한국 반도체 밸류체인 동향
  - 환율(KRW/USD) 영향 분석

## 4. 경쟁 장벽 분석

### 4.1 CrossFin의 독점적 강점

| 경쟁 장벽 | 설명 | 진입 난이도 |
|----------|------|-----------|
| **한국어 데이터 파싱** | 네이버 금융, KRX 등 한국어 소스 파싱/정규화 역량 | 높음 — 한국어+금융 도메인 지식 필요 |
| **외인/기관 수급 데이터 영어 제공** | 전 세계에서 이 데이터를 영어 API로 저렴하게 파는 곳 없음 | 매우 높음 — 유일한 경쟁자 TradePulse는 $10K/월 |
| **실시간 뉴스 번역** | 한국 주식 뉴스를 실시간 영어 번역 | 중간 — AI 번역 비용 존재하나 진입 가능 |
| **MCP/A2A 통합** | AI 에이전트가 직접 한국 주식 데이터 쿼리 가능 | 높음 — CrossFin은 이미 MCP 서버 보유 |
| **가격 경쟁력** | TradePulse $10K vs CrossFin $499 | 지속 가능 — CrossFin의 비용 구조가 가벼움 |

### 4.2 잠재 경쟁자 & 진입 난이도

| 잠재 경쟁자 | 위협 수준 | 설명 |
|-----------|----------|------|
| **글로벌 데이터 업체 (Polygon, EODHD 등)** | 중간 | 한국 시세 데이터는 추가할 수 있으나, 수급/뉴스 번역은 별도 투자 필요 |
| **KRX 직접** | 낮음 | 글로벌 개발자 친화적 API를 만들 동기가 약함 |
| **TradePulse** | 중~높음 | 이미 수급 데이터 제공 중이나, 가격이 월 $10K로 높음 |
| **한국 핀테크 스타트업** | 중간 | 한국어 역량 있으나 글로벌 마케팅/영어 부족 |
| **Bloomberg/Refinitiv** | 낮음 | 이미 제공하나 터미널 비용 $25K+/년, API 별도 |

### 4.3 핵심 방어 전략

1. **선점 효과**: 영어 한국 주식 수급 데이터 API 시장을 먼저 장악
2. **가격 파괴**: TradePulse 대비 95% 저렴한 가격으로 시장 진입 장벽 구축
3. **AI 에이전트 생태계**: MCP 서버를 통한 AI 에이전트 통합 — 독점적 채널
4. **커뮤니티 빌딩**: 한국 주식 투자 외국인 커뮤니티 (Reddit, Discord) 육성
5. **데이터 범위 확대**: 시세 → 수급 → 뉴스 → 공시 → 재무 → 컨센서스 순차 확장

## 5. 실행 로드맵

### Phase 1 (1~3개월): MVP 출시
- KOSPI/KOSDAQ 종목 시세 API (일봉/주봉)

- 뉴스 전문 AI 번역

### Phase 3 (7~12개월): Enterprise & 확장

- 헤지펀드 직접 영업

**"영어로 한국 주식 외인/기관 수급 데이터를 저렴하게 API로 제공하는 곳은 세계에 없다."**
