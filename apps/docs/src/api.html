API Reference (Core)

Routing

  • GET /api/route/status - exchange network health
  • GET /api/route/pairs - bridge pair availability
  • GET /api/route/fees - fee and transfer table
  • GET /api/premium/route/find - full paid route planner

Arbitrage

  • GET /api/arbitrage/demo - free top-3 decisions
  • GET /api/premium/arbitrage/opportunities - paid full scan

ACP

  • GET /api/acp/status
  • POST /api/acp/quote
  • POST /api/acp/execute (simulation)
