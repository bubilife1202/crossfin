Overview Quickstart API MCP crossfin.dev

API Reference

Base URL: https://crossfin.dev — 7 exchanges: Bithumb, Upbit, Coinone, GoPax, Binance, OKX, Bybit

Routing (Free)

  • GET /api/route/exchanges — supported exchanges and coins
  • GET /api/route/status — exchange network health (7/7 online)
  • GET /api/route/pairs — bridge pairs with live prices
  • GET /api/route/fees — trading + withdrawal fee table

Routing (Paid, x402)

  • GET /api/premium/route/find — full optimal route analysis ($0.10)

Arbitrage & Kimchi Premium

  • GET /api/arbitrage/demo — free top-3 decisions (EXECUTE/WAIT/SKIP)
  • GET /api/premium/arbitrage/kimchi — full 11-pair kimchi premium ($0.05)
  • GET /api/premium/arbitrage/opportunities — full arbitrage scan ($0.10)
  • GET /api/premium/arbitrage/kimchi/history — 7-day history ($0.05)

Korean Market Data

  • GET /api/premium/market/fx/usdkrw — USD/KRW rate ($0.01)
  • GET /api/premium/market/upbit/ticker — Upbit ticker ($0.02)
  • GET /api/premium/market/upbit/orderbook — Upbit orderbook ($0.02)
  • GET /api/premium/market/coinone/ticker — Coinone ticker ($0.02)
  • GET /api/premium/bithumb/orderbook — Bithumb orderbook ($0.02)
  • GET /api/premium/crypto/korea/5exchange — cross-exchange prices ($0.08)
  • GET /api/premium/news/korea/headlines — Korean crypto news ($0.03)

Korean Stock APIs

  • GET /api/premium/market/korea/indices — KOSPI & KOSDAQ ($0.03)
  • GET /api/premium/market/korea/stocks/momentum — top movers ($0.05)
  • GET /api/premium/market/korea/investor-flow — investor flow ($0.05)
  • GET /api/premium/market/korea/etf — 1,070+ ETFs ($0.03)

Bundles

  • GET /api/premium/morning/brief — Morning Brief ($0.20)
  • GET /api/premium/crypto/snapshot — Crypto Snapshot ($0.15)
  • GET /api/premium/kimchi/stats — Kimchi Stats ($0.15)
  • GET /api/premium/market/korea/stock-brief — Stock Brief ($0.10)

ACP (Agent Commerce Protocol)

  • GET /api/acp/status — ACP protocol status
  • POST /api/acp/quote — routing quote (free preview)
  • POST /api/acp/execute — execute simulation

Registry & Discovery (Free)

  • GET /api/registry — all services
  • GET /api/registry/search?q= — full-text search
  • GET /api/registry/categories — categories with counts
  • GET /.well-known/crossfin.json — agent auto-discovery
  • GET /api/openapi.json — OpenAPI 3.1 spec
  • GET /api/docs/guide — structured agent guide
