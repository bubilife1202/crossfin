MCP Integration

16 tools for Claude Desktop or any MCP client. Routes across 7 exchanges with 11 bridge coins.

Install

npx -y crossfin-mcp

npm: crossfin-mcp

Claude Desktop config

{
  "mcpServers": {
    "crossfin": {
      "command": "npx",
      "args": ["-y", "crossfin-mcp"],
      "env": {
        "EVM_PRIVATE_KEY": "0x..."
      }
    }
  }
}

No EVM key? Free tools work without one. Paid tools require a Base wallet with USDC.

All 16 Tools

  • find_optimal_route — optimal path across 7 exchanges ($0.10)
  • list_exchange_fees — trading + withdrawal fees (free)
  • compare_exchange_prices — live price comparison (free)
  • get_kimchi_premium — Korean vs global spread (free)
  • call_paid_service — call any paid API with x402
  • search_services — search 184+ services (free)
  • list_services — browse service catalog (free)
  • get_service — service details (free)
  • list_categories — service categories (free)
  • get_guide — full agent guide (free)
  • get_analytics — gateway usage stats (free)
  • create_wallet — local ledger wallet (free)
  • get_balance — check wallet balance (free)
  • transfer — transfer between wallets (free)
  • list_transactions — transaction history (free)
  • set_budget — daily spend limit (free)
