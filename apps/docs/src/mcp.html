MCP Integration

Use CrossFin tools in Claude Desktop or any MCP client.

Install

npx -y crossfin-mcp

Claude Desktop config

{
  "mcpServers": {
    "crossfin": {
      "command": "npx",
      "args": ["-y", "crossfin-mcp"],
      "env": {
        "EVM_PRIVATE_KEY": "0x..."
      }
    }
  }
}

  • get_kimchi_premium
  • list_exchange_fees
  • compare_exchange_prices
